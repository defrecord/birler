* defrecord Logger

** Goals

Simple logging service for predictive analytics and error detection.

** Installation

This can be consumed as a stand-along service.

#+BEGIN_SRC html
  <script src="defrecord.logger.js"></script>
  <div id=""></div>

#+END_SRC

** Development

    lein figwheel

** Testing

    lein test

** Building

    lein cljsbuild once

** Deploying

    lein essthree
